========================================
INSTALADOR DE DESCARGASORDENADAS
========================================

Este directorio contiene los scripts necesarios para crear e instalar la aplicación DescargasOrdenadas.

INSTRUCCIONES DE USO:

1. PARA INSTALAR LA APLICACIÓN:
   - Ejecute "Instalar.bat" haciendo doble clic.
   - Si aparece un diálogo de seguridad, seleccione "Sí" para permitir los cambios.
   - Siga las instrucciones en pantalla para completar la instalación.
   - Puede elegir dónde instalar la aplicación y se crearán accesos directos automáticamente.

2. PARA CREAR EL INSTALADOR (DESARROLLADORES):
   - Ejecute "crear_instalador.bat" para compilar la aplicación y preparar los archivos necesarios.
   - Se creará una carpeta "instalador" con todo lo necesario para distribuir la aplicación.
   - La carpeta "instalador" puede compartirse con los usuarios finales.

3. SCRIPTS INCLUIDOS:
   - Instalar.bat: Inicia el proceso de instalación con privilegios de administrador.
   - setup.bat: Realiza la instalación propiamente dicha (copia archivos, crea accesos directos).
   - crear_instalador.bat: Compila la aplicación y prepara los archivos para distribuir.

Si encuentra algún problema durante la instalación, por favor contacte con el soporte técnico.
